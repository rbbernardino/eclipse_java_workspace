Tutorial GraphG


Introdução


O GraphG é um programa usado para criar grafos de estrutura de dados usando uma interface agradável e bonita e converter para o formato .DOT


Barra de ferramentas


  • Novo – Cria um novo arquivo .*GRG

  • Abrir – Abre um arquivo *GRG salvo

  • Salvar – Salva o grafo desenhado no formato *GRG

  • Exportar – Exporta o grafo desenhado para o formato *.DOT


Barra de desenho


  • Selecionar – Seleciona os objetos do grafo

  • Vértice – Desenha uma vértice na área do grafo

  • Aresta – Liga as vértices do grafo


Como desenhar um grafo


Selecione na barra de desenho o botão de vértice, em seguida clique na área de desenho para criar as vértices.

Selecione na barra de desenho o botão de aresta e em seguida clique no primeiro vértice e arraste até o segundo vértice e solte o botão do mouse

Selecione na barra de desenho o botão de selecionar para selecionar os objetos do grafo para alterar a posição no grafo.


Salvar o grafo


O GraphG é um programa que salva os grafos inicialmente no formato *.GRG.

Para salvar basta clicar no botão Salvar na barra de ferramentas, no menu Arquivo>Salvar ou usar a tecla de atalho CTRL+S


Abrir um arquivo


Para abrir um arquivo *.GRG basta clicar no botão Abrir na barra de ferramentas, no menu Arquivo>Abrir ou usar a tecla de atalho CTRL+O


Criar um novo arquivo


Para criar um novo arquivo clique no botão Novo na barra de ferramentas, no menu Arquivo>Novo ou use a tecla de atalho CTRL+N


Exportar um grafo


Para exportar um grafo clique no botão Exportar na barra de ferramentas, no menu Arquivo>Exportar ou use a tecla de atalho CTRL+E


Para sair do programa basta clicar em Arquivo>Sair ou usar a tecla de atalho ALT+F4.
